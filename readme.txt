Плагин добавляет дополнительные настройки на страницу Курсы созданную с помощью плагина WP Courseware и WooCommerce.

== Описание ==

Официальный плагин AB Inspiration для WooCommerce и WP Courseware.

На странице настроек плагина вы можете выбрать какие курсы скрыть со страницы Каталога курсов, отобразить цену товара Woocommerce в описании курса и подключить лендинг к курсу, вместо стандартной страницы курса.

= Настройка плагина =

1. Перейдите в настройки плагина Дополнительные настройки
2. Отметьте галочкой те курсы, которые не должны отображаться на странице Курсы (если необходимо)
3. Выберите из списка товары для каждого курса
4. Выберите из списка лендинги для каждого курса (если вы создали лендинги)
5. Сохраните настройки


= Disclaimer =
Этот плагин не является плагином eCommence или LMS. Это плагин с дополнительными настройками для сайта онлайн школы созданном с помощью плагинов Wp Courseware и WooCommerce.

== Установка ==

1. Загрузите `Addon for AB-Inspiration, WooCoocommerce and WP Courseware` папке в `/wp-content/plugins/` директорию
1. Активируйте плагин в разделе 'Плагины' в админке сайта

== Перевод плагина ==

Плагин локализован. Сейчас в плагине доступны 2 языка: Русский и Английский. Если ваш сайт не на русском и не на английском языках, вы можете самостоятельно сделать перевод плагина. В папке wpcourseware-extra-settings/languages вы найдете файл в расширением .pot. Откройте его в программе poedit и создайте перевод. Сохраните файл с названием wpcoursewarextra-ru_RU.po (вместо ru_RU укажите индекс языка вашего языка. Узнать индекс вашего языка можно здесь: https://translate.wordpress.org/ )
